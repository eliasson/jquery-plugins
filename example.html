jquery.palette

Click the input field to see the palette, the color value will be written as RGB to the elements attribute data-colorpicker-value. And the elements background color will be set to the same color.

You could also bind it to other elements, like an label:

Disable palette

If you for some reason want to disable the color palette, just set the attribute data-disabled to true. To try this, click on the button and see what happens if you try click either on the label or the input field.
